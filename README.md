# romen
Romen – это ресторан, который находится всего в 45 км от Кишинёва, в городе Оргеев.
Ресторан специализируутся на национальной молдавской кухне. 
## Project setup
- git clone http://github.com/S-Anzhela/romen.git
- cd romen
- npm i
- npm run serve
## Technologies
- Vue
- Node
- HTML
- CSS
- JavaScript
